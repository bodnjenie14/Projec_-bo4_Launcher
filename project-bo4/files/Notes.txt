## **Update Notes**

### **General Features**
- **Bots with Random Skins and Weapon Camos/Attachments**: Bots in MP can now use random weapons, skins, and camos.
- **Reactives, Camos, and Calling Cards**: Now fully functional and integrated.
- **Extra Outfits in MP**: Numbers outfits and some unused are working again (can reset sometimes).
- **Unlock Settings Saving**: Unlock settings are now saved automatically.
- **Version Number Display**: Version number is now visible in the UI.
- **External Console**: an External Console to show logs and input commands.

### **Blackout **
- **Map Selection**: Choose your Blackout map using the new Lua menu.
- **Unused Customizable Prestige Icons**: Restored and fully working unused prestige icons in Blackout.

### **User Interface**
- **UI Coloring**: Customize UI colors directly from the shield menu.gelonade 
- **Unlock Menu**: The unlock menu has been reworked and renamed to **Shield's Menu.**
- **New Error Messaging System**: Message errors are easier to understand.

### **Server Browser**
- **Server Browser Added**: Browse servers easily (note: lobby browser is not yet supported).
- **Custom IP and Username Input**: Enter custom IP addresses in Server Browser and set username in Shield's Menu.
- **DW Server Button**: Button to select servers when you get disconnected.

### **Prestiging**
- **Prestige Master**: Introduced Prestige Master, complete with set ranks and a dedicated button.

### **Fixes**
- **XUID Rework**: XUID system reworked.
- **Black Screen Issue**: Resolved black screen problems.
- **Multiple Instances**: Fixed issues with running two instances simultaneously.
- **Specialist HQ Mode**: Fixed bots not being added and black screen issue in Specialist HQ mode (mostly training parts).
